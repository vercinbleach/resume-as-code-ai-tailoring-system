after joining Grup Montaner as a Software Engineer in July 2025.

LinkedIn currently shows one uninterrupted `Software Engineer` role from July
2025 to the present. The January 2026 promotion is retained because it was
confirmed directly by Vincenzo; the LinkedIn title has not yet been updated.

Work on the Operations App spans the transition between the Software Engineer
and Lead Software Engineer titles. The implementation may be grouped with the earlier
role in a one-page resume, while ongoing team direction remains part of the
Lead Software Engineer scope.

## Operational Process Digitalization Project

- Architected and led the full-stack development of a critical Business Process
  Management platform for an organization with EUR 90M in annual revenue, using
  an open-source Java engine and a Next.js/React frontend.
- Built a central hub that orchestrates high-stakes operations across core
  departments, improving data consistency, process reliability, and
  communication.
- Designed a scalable Java/Python microservices ecosystem for CRM
  synchronization, legal sanction tracking, and ISO compliance ID generation.
- Implemented a hybrid RPA integration layer for an API-less legacy ERP,
  centralizing data across business processes and increasing departmental
  productivity.
- [deprecated] Implemented executive Power BI dashboards with centralized
  strategic KPIs for department heads and executive leadership. This remains a
  historical contribution, but Power BI is no longer the current reporting
  approach.
- [current] Connects an AI agent to the operational data so users can ask
  questions and hold conversations with the data instead of relying on static
